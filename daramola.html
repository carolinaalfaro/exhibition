A Timely Solution

When working on the timely solution project in History, we had to create many conponents that contribute to it. First, we had to chose an Industrial Revolution invention, in this case it we picked the locomotive and argue that it was best. To do this we had to find articals based on the inventing and write summary. Not only that, but we also included a sketch, 3D model, and a powerpoint to sum it all up.
